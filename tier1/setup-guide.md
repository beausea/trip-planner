# 플랫폼별 설정 가이드

설치 없이 바로 사용할 수 있는 방법을 소개합니다.
모두 무료 계정으로 시작할 수 있습니다.

---

## 방법 1: Claude.ai Projects (추천)

### 준비물
- Claude.ai 계정 (무료 가입: [claude.ai](https://claude.ai))

### 설정 순서

**1단계 — 새 프로젝트 만들기**

1. [claude.ai](https://claude.ai) 로그인
2. 왼쪽 사이드바 → **"Projects"** 클릭
3. **"+ New project"** 클릭
4. 프로젝트 이름 입력: `가족 여행 플래너`

**2단계 — 시스템 프롬프트 설정**

1. 프로젝트 생성 후 **"Set project instructions"** 클릭
2. [`claude-project-prompt.md`](claude-project-prompt.md) 파일을 텍스트 편집기로 열기
3. `---` 구분선 **아래** 내용 전체를 선택 → 복사 (Ctrl+C)
4. Claude.ai 지침 입력란에 붙여넣기 (Ctrl+V)
5. **"Save"** 클릭

**3단계 — 사용 시작**

프로젝트 채팅에서 바로 입력하세요:

```
/plan-trip 5월 경주 2박3일 초등5학년 아이와 함께
```

### 무료 vs 유료 차이

| 항목 | 무료 | Pro (월 $20) |
|------|------|-------------|
| 사용 한도 | 일정 메시지 수 제한 | 5배 이상 더 많음 |
| 모델 | Claude 3.5 Sonnet | Claude 3.5 Sonnet (우선 접근) |
| 프로젝트 파일 | 최대 5개 | 최대 20개 |

> 💡 **무료로도 충분합니다.** 여행 계획 1회 작성은 무료 한도 내에서 가능합니다.

## 방법 2: Gemini Gems (Google — 무료 한도 가장 넉넉함)

### 준비물
- Google 계정 (기존 Gmail 계정 사용 가능)

### 설정 순서

**1단계 — Gemini 접속**

1. [gemini.google.com](https://gemini.google.com) 접속
2. Google 계정으로 로그인

**2단계 — Gem 만들기**

1. 왼쪽 메뉴 → **"Gems"** 클릭
2. **"새 Gem 만들기"** (또는 "Create a Gem") 클릭
3. Gem 이름: `가족 여행 플래너`

**3단계 — 지침 설정**

1. [`gemini-gems-prompt.md`](gemini-gems-prompt.md) 파일을 텍스트 편집기로 열기
2. `---` 구분선 **아래** 내용 전체를 복사
3. Gem의 "지침" (Instructions) 입력란에 붙여넣기
4. **"저장"** 클릭

**4단계 — 사용 시작**

저장한 Gem을 열고 입력하세요:

```
/plan-trip 5월 경주 2박3일 초등5학년 아이와 함께
```

### Gemini의 장점

- **Google 검색 연동**: 실시간 운영 정보, 최신 행사 확인에 강함
- **무료 한도 넉넉함**: 개인 사용자 기준 하루 1M 토큰 (매우 넉넉)
- **Google 계정 활용**: 별도 가입 불필요

## 방법 3: ChatGPT (OpenAI)

### 준비물
- ChatGPT 계정 (무료 가입: [chat.openai.com](https://chat.openai.com))

### 방법 A — 매번 붙여넣기 (무료, 간단)

1. ChatGPT 새 채팅 시작
2. `claude-project-prompt.md`의 `---` 구분선 아래 내용을 먼저 붙여넣기
3. 같은 메시지에 이어서: `/plan-trip 여행조건` 입력

> ⚠️ ChatGPT 무료는 대화마다 새로 붙여넣어야 합니다.

### 방법 B — Custom GPT (Plus 구독 필요, 월 $20)

1. ChatGPT → 좌측 메뉴 **"Explore GPTs"** → **"Create"**
2. "Instructions" 탭에 프롬프트 내용 붙여넣기
3. GPT 저장 후 재사용 가능

## 자주 묻는 질문

### Q: 어떤 플랫폼이 제일 좋나요?

| 상황 | 추천 |
|------|------|
| 처음 써보는 분 | **Claude.ai** (인터페이스가 가장 직관적) |
| 비용이 걱정되는 분 | **Gemini** (무료 한도 가장 넉넉) |
| Google 계정만 있는 분 | **Gemini** (추가 가입 불필요) |
| 최신 정보가 중요한 분 | **Gemini** (Google 검색 연동) |
| 이미 Claude Pro 쓰는 분 | **Claude.ai** (Projects 기능 활용) |

### Q: 한 번 사용하는 데 얼마나 걸리나요?

처음 설정: 약 5분
여행 계획 생성: 약 10~20분 (대화 몇 번)

### Q: 생성된 계획서는 어떻게 저장하나요?

AI가 생성한 내용을 **복사(Ctrl+A → Ctrl+C)** 해서 메모장이나 한글/Word에 붙여넣으세요.
마크다운 형식이라 `Notion`, `Obsidian` 같은 노트앱에서도 그대로 사용할 수 있습니다.

### Q: 운영 시간, 입장료가 틀릴 수도 있나요?

네, AI는 최신 정보를 놓칠 수 있습니다.
계획서에 나오는 운영 시간과 입장료는 반드시 **방문 전날 공식 홈페이지나 전화로 재확인**하세요.

### Q: API 키나 특별한 설정이 필요한가요?

**Tier 1 (이 폴더)는 API 키가 전혀 필요 없습니다.**
계정만 만들면 바로 사용할 수 있습니다.

## 문제가 생겼을 때

| 증상 | 해결 방법 |
|------|----------|
| AI가 한국어로 답하지 않음 | 프롬프트 맨 위에 "모든 답변은 한국어로 해주세요" 추가 |
| 여행 계획이 너무 빡빡함 | `/retry` 입력 후 "여유롭게 짜줘"라고 추가 요청 |
| 교육 자료가 너무 어려움 | "초등 3학년 수준으로 다시 써줘"라고 요청 |
| 비용 계산이 이상함 | `/budget` 입력 후 구체적인 조건 다시 설명 |
| 응답이 중간에 잘림 | "계속해줘" 입력 |
